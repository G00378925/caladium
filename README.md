# Caladium

Caladium is my fourth year software project, the title of my dissertation is "Malware detection using dynamic heuristic analysis".

Caladium is platform contains three subprojects, a Windows GUI application, service side backend, and the analysis microservices,<br>
technicals will have the ability to setup analysis computers, and add them to networking using the dashboard SPA.

- In `/client` you will find the code for the GUI Tkinter application.

- `/server` contains the Flask server and SPA dashboard.

- `/sandbox` contains the sandbox code, and syscall analysis.
